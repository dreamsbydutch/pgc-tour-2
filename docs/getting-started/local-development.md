# Local Development

This repo is a TanStack Start app (Vite) with a Convex backend.

## Prerequisites

- Node.js: `20.x` (see `package.json -> engines.node`)
- npm
- Accounts/keys as needed:
  - Convex
  - Clerk
  - PostHog

## Install

```bash
npm install
```

## Run (frontend)

```bash
npm run dev
```

- Vite runs on `http://localhost:3000` (see `package.json` script).

## Run (Convex)

In a second terminal:

```bash
npm run convex:dev
```

Notes:

- The first time, prefer `npx convex dev --configure` to create/configure a Convex project and populate `.env.local`.
- Convex generates types under `convex/_generated/*`.

## Typical workflow

1. `npm run dev` for the app.
2. `npm run convex:dev` for backend functions and schema.
3. When you add/remove Convex exports, regenerate types (Convex dev does this automatically while running).

## Useful checks

- Typecheck: `npm run typecheck`
- Test: `npm run test`
- Lint: `npm run lint` (use `npm run lint:strict` in CI-style situations)
- Format: `npm run format`

## Repo "generated" files

Do not hand-edit:

- `convex/_generated/*`
- `src/routeTree.gen.ts`

These are generated by Convex and TanStack Router tooling.
